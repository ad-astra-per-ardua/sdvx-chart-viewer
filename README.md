# SDVX Megamix-info

SOUND VOLTEX 채보중 메가믹스에 대응하는 곡들의 정보를 제공

---

## 주요 기능

### 곡 목록 (`/`)
- 전체 수록곡을 목록으로 표시
- **검색**: 제목 · 아티스트 · 키워드로 실시간 검색
- **레벨 필터**: 최소~최대 레벨 범위 지정 (1 ~ 20.9)
- **레벨 퀵바**: 정수 레벨 한 칸씩 빠르게 선택
- **난이도 필터**: NOV / ADV / EXH / MXM / INF / GRV / HVN / VVD / XCD / ULT / NBL
- **태그 필터**: 등록된 태그로 채보 분류 검색
- **정렬**: 최신순 · 레벨 높은순 · 레벨 낮은순

### 채보 상세 (`/charts/:id`)
- 곡 재킷, 제목, 아티스트, 전체 난이도 배지 표시
- 난이도 배지 클릭 시 해당 채보로 이동
- 채보 이미지를 **파트별**로 구분해서 표시
  - 인트로 / 메인파트 / 대체파트 / 아웃트로
- 이미지 클릭 시 **라이트박스** 확대 보기 (재클릭 또는 `Esc`로 닫기)
- 등록된 태그 목록 표시

### 메가믹스 선곡 리스트 (`/megamix`)
- **자선곡 5곡 + 타선곡 5곡** 슬롯 구성
- 제목·아티스트 검색으로 곡 추가
- 슬롯 드래그 앤 드롭으로 순서 변경
- 선택 내용은 브라우저에 자동 저장 (새로고침 후에도 유지)
